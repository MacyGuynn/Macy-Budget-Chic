I created my own repository and then made my first HTML page. After creating the page I learned numerous different tags and what they ment such as the one I am using now (p) which is a paragraph tag. After this I found out how to create flexboxes and how they shift depending on the technology someone is viewing from. I then began creating a CSS page which curates the style of the website like the colors, fonts, photos, etc. In my CSS code, I focused on styling elements by using properties like color, font-family, and spacing to make the website look more polished and consistent. I also used things like nav to justify layout features in flexbox along with padding and alignment to control where content appears on the page and make it more visually balanced. Overall I learned so much from this course and am excited to use my new skills in the future. I created this website for female college students like myself who love fashion but need to Budget. I wanted to provide girls with cute clothing options that are afordable but still durable. The collections are based off of clothes I have that I know are high quality and still cheap.
